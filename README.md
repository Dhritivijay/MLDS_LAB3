Small Image Classification Using Convolutional Neural Network (CNN) – Hyperparameter Tuning 📌 Overview

This project demonstrates image classification using a Convolutional Neural Network (CNN). The model is trained on a small image dataset to understand how different hyperparameters influence training performance, accuracy, and loss.

🎯 Objective

Implement a CNN model for image classification

Understand the role of hyperparameters such as epochs, optimizer, and learning rate

Evaluate model performance using accuracy and loss metrics

🧠 Model Architecture

The CNN model consists of:

Input Layer (Image Data)

Convolutional Layers for feature extraction

ReLU Activation Function

Pooling Layers for dimensionality reduction

Fully Connected Dense Layer

Output Layer with Softmax Activation

⚙️ Technologies Used

Python

TensorFlow / Keras

Google Colab

NumPy

Matplotlib

🚀 Steps Performed

Imported required libraries

Loaded and preprocessed the dataset

Built the CNN architecture

Compiled the model with optimizer and loss function

Trained the model and monitored accuracy/loss

Evaluated performance on test data

📊 Results

The CNN model successfully classified images and achieved good accuracy after training. Training and validation graphs were used to analyze performance and convergence.
